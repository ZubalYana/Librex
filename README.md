# Librex — Book Exchange Service

A web app where users can list their own books, browse and search books added by others, view book details, and send exchange requests via email.

Built as a solution to the following technical assignment:

> Develop a web app where users can add their own books, search other users' books, view book details, and send exchange requests.

## Tech Stack

- **Frontend:** React, TypeScript, TailwindCSS, React Router, Zustand
- **Backend:** Node.js, Express, TypeScript
- **Database:** PostgreSQL via Prisma ORM
- **Auth:** JWT
- **Email:** Resend
